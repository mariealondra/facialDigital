El proyecto abarcó tres etapas generales para su aplicación. Estas etapas son: página web para el terapista, aplicación móvil para el usuario, y dispositivo para la captura de señales electromiográficas. Existieron errores de lectura en la señal a causa de conexiones en el circuito, pero luego se logró capturar las señales de electromiografía mediante una conexión completa del circuito.
